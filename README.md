# Hotosevents Website

## How to View
Since this is a static website (HTML/CSS/JS), you do not need to install any dependencies.

### Option 1: Direct File Open (Easiest)
Simply double-click `index.html` in your file explorer, or drag it into your web browser.

### Option 2: Local Server (Recommended for best experience)
If you have Python installed, you can run a smooth local server:

1. Open your terminal in this folder.
2. Run:
   ```bash
   python3 -m http.server
   ```
3. Open `http://localhost:8000` in your browser.

## File Structure
- `index.html` - Home Page
- `redlock.html` - RedLock Division
- `corporate.html` - Corporate Division
- `css/` - Styles
- `js/` - Scripts
